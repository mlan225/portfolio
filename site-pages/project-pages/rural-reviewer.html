Litter Node Project

Node.Js | Mongodb | Javascript | HTML | CSS | async

This application will allow you to control your own, currently watered-down version of Facebook or Twitter. A user will be able to:

  1. Make an account
  2. Follow/unfollow other users to view their posts on the news feed
  3. Edit profile Information

Something I have added was the ability for users to manually submit bug reports and request features. These feature requests then show up on the admin panel where I can manage the requests and even create moderator posts that display with a special styling on a user's newsfeed.

As I mentioned, I have planned to continue developing this application, adding features that will make the user experience more efficient and clean all while learning several new tools and packages as I further develop Litter.

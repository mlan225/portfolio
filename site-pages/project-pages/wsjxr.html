The Wall Street Journal Exchange Rates (WSJXR)

Node.Js | Javascript | Google API | JSON

This application is a tool to help keep track of the exchange rates presented on the Wallstreet Journal's site. The project was built within the Node.js framework and uses the Google API to record and reflect the JSON data on the WSJ website within an excel sheet. The JSON is received and formatted before sending it to the view using Javascript.

On the WSJXR site, there is a link to view both the WSJ page with the original information and one to go straight to the google sheet. Any changes made in the google sheet will be overwritten upon loading the WSJXR site. This allows it to stay up to date and remove any edits previously made from other users that have run queries on the information.

This project is currently undergoing changes and cannot be viewed as Google OAuth requires changes to the previous authentication methods.
